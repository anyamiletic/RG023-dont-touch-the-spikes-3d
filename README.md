# RG023-dont-touch-the-spikes-3d

> bounce from wall to wall, collect tokens, and try not to touch the spikes!

## Requirements

```sh
OpenGl
```

## Compiling

```sh
$ make
```

## Gameplay

```sh
[https://youtu.be/jFnMSaCGJCU](youtube link)
```


## Controls

| Command       | Function      |
| ------------- | ------------- |
| G             | normal mode   |
| C             | cave mode     |
| Space         | Jump          |
| S             | Pause         |
